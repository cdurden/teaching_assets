This is called a system of two equations.

The solution to the system of equations is: $$x=6,y=4$$

This solution can be written using an ordered pair: $$(x,y) = (6,4)$$

$x = $ weight of the cat
$y = $ weight the rabbit
$x+y = 10$
$y = 4$

The cat weighs $10-4=6$ units.

The rabbit weighs $4$ units.
