Remember that a compound inequality is two inequalities that are joined together using the word "and" or "or".
The first number line represents all numbers $x$ for which $x < 5$.
The second number line represents all numbers $x$ for which $x >-2$.

Inequality

$x < 5$

$x > -2$

This number line represents all numbers $x$ for which $x < 5$ and $x > -2$.
